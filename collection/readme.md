# 铜の币纪主站

一个功能完整的纸币、硬币收藏品管理与展示平台。

---

## 功能概览

### 1. 纸币与硬币双板块

底部 Tab 栏有「纸币」和「硬币」两个主板块，各自拥有独立的：

- **分类侧边栏** — 按类别（纪念钞、人民币、港币、外币、纪念币、流通币、金银币等）组织藏品，支持点击展开/折叠子分类
- **概览页** — 展示该板块全部藏品的总览，按分类分组，每件藏品可查看图片、名称、冠字号、年份、评级、目录编号等信息
- **分类内容页** — 以系列（Series）为单位的折叠列表，点击可展开查看该系列下的具体品种（Varieties）和每一件藏品（Copies）的详细信息
- **独立滚动容器** — 每个分类视图、概览视图、搜索结果视图都有各自的滚动容器，切换后回来时**滚动位置和展开状态完好保留**

### 2. 搜索功能

支持纸币和硬币两个板块的全文搜索，提供两种搜索模式：

- **实时搜索**（默认）— 输入即搜索，搜索结果实时更新
- **点击搜索** — 输入关键词后手动点击"搜索"按钮

搜索字段支持：
- **全字段搜索** — 在所有内容中匹配
- **按名称搜索** — 匹配系列名称、品种名称
- **按冠字号搜索** — 匹配版本号/冠字号
- **按年份搜索** — 匹配发行年份
- **按评级机构搜索** — 匹配评级分数或机构名称
- **按目录编号搜索** — 匹配 Pick# 或 KM# 编号（自动补全前缀）

搜索结果按分类分组，每件藏品可点击跳转到其所在系列位置，并自动展开到对应条目。

### 3. 图片查看器

点击藏品图片可打开全屏查看弹窗，支持：
- **双指捏合缩放**（1~4倍）
- **拖拽平移**（放大后）
- **双击重置**（恢复原始大小和位置）
- **点击背景关闭**
- 正反面双图快速切换

### 4. 文章阅读器

收集纸币和硬币数据中所有带有 `readme` 的系列或品种，形成完整的文章索引：

- **分类侧边栏** — 按纸币/硬币分类统计文章数量，支持按分类筛选
- **文章列表** — 全部分类或当前分类下的文章，按来源分组显示，可看到文章路径
- **全文搜索** — 支持**按标题索引**（实时匹配标题）和**全字段索引**（需预加载后，可搜索正文内容，带上下文片段预览）
- **内容高亮** — 搜索结果中的关键词在标题和正文片段中高亮标记
- **阅读器** — 点开文章后**每次从顶部开始阅读**，支持加载缓存的 HTML 内容，图片路径自动适配
- **独立容器** — 每篇文章独立容器，返回列表时**列表滚动位置保留**

### 5. 专题收藏

通过 `fun-data-bridge.js` 和 `special-bridge.js` 接入趣味收藏数据（如"年份图鉴"等专题）：

- **专题卡片概览** — 展示所有专题的名称和藏品数量
- **专题详情页** — 按条目展示，每件藏品可点击查看大图
- 支持子分类侧边栏（如果专题配置了分类）
- 支持自定义图片路径

### 6. 统计与数据分析

在我的（设置）页面中提供丰富的统计功能：

**藏品数量统计：**
- 藏品总数量、纸币数量、硬币数量、已记录价格数量

**资金投入统计：**
- 藏品总投入金额、藏品均价
- **价格列表** — 所有藏品的价格明细，支持按分类筛选、按价格排序（从高到低/从低到高），显示每个板块的总投入和均价

**评级得分统计：**
- 按纸币/硬币分别统计评级分布（柱状图展示）
- 支持切换查看纸币和硬币各自的评级分布
- 未评级藏品单独统计

**藏品年代统计：**
- 按十年份区间统计藏品数量（柱状图展示）

### 7. 数据导出

提供四种数据导出格式：

| 格式 | 内容 |
|---|---|
| **JSON** | 完整数据结构备份，包含类型、分类、系列、冠字号、年份、评级、目录编号、价格、购买日期、材质、备注 |
| **CSV** | 详细表格（UTF-8 BOM 编码，Excel 可直接打开），按行展示每件藏品的完整信息 |
| **Markdown** | 概览报告，包含总览、按分类统计的数量和小计金额 |
| **TXT** | 价格清单，按从高到低排列所有藏品的价格 |

### 8. 主题色自定义

- 预设 5 种主题色（蓝、红、绿、橙、紫）
- 支持**自定义颜色**，使用取色器选取后添加
- 最多保存 20 个自定义颜色
- 主题色自动调整边框、背景、侧边栏等关联颜色

### 9. 侧边栏折叠

点击左上角按钮可折叠/展开侧边栏，状态在纸币和硬币板块之间独立保存。

### 10. 数据桥接

所有数据文件（纸币、硬币、趣味收藏）来自独立的旧站数据仓库，通过桥接文件（`data-bridge.js`、`coin-data-bridge.js`、`fun-data-bridge.js`、`special-bridge.js`）统一收集到 `window.DATA_MAP`、`window.COIN_DATA_MAP`、`window.FUN_DATA_MAP` 中，供主站使用。分类树定义在 `config.js`（纸币）和 `coin-config.js`（硬币）中，可灵活扩展。

## 技术特性

- **独立滚动容器系统** — 每个视图（概览、每个分类、搜索结果、文章列表、每篇文章）都有独立的滚动容器，切换后位置和状态完全保留
- **哈希表缓存** — 文章 HTML 内容和纯文本内容缓存，二次加载无需重新请求
- **懒加载渲染** — 切换到已有容器的视图时，检测到已有内容则跳过重新渲染，仅恢复滚动位置和展开状态
- **Hammer.js 手势库** — 图片查看器支持捏合缩放和拖拽平移

## 页面结构

| Tab | 功能 |
|---|---|
| 纸币 | 纸币收藏品概览、分类浏览、搜索 |

## 数据来源

所有数据文件位于独立的仓库目录中：
- `../notecollection/` — 纸币数据（纪念钞、连体钞、人民币、港币、澳门币、台币、外币、民国纸币、票证等）
- `../coincollection/` — 硬币数据（纪念币、流通币、金银币）
- `../funcollection/` — 趣味收藏数据
