(function() {
    function initGkqAccordions() {
        var accordions = document.querySelectorAll('.rich-content .gkq-accordion');
        for (var i = 0; i < accordions.length; i++) {
            var accordion = accordions[i];
            if (accordion._bound) continue;
            accordion._bound = true;
            
            var header = accordion.querySelector('.gkq-accordion-header');
            var content = accordion.querySelector('.gkq-accordion-content');
            var icon = header.querySelector('.gkq-accordion-icon');
            var inner = content.querySelector('.gkq-accordion-content-inner');
            
            if (!content || !inner) continue;
            
            content.style.height = '0';
            
            header.addEventListener('click', function(e) {
                e.stopPropagation();
                var self = this.parentElement;
                var c = self.querySelector('.gkq-accordion-content');
                var ic = this.querySelector('.gkq-accordion-icon');
                var inn = c.querySelector('.gkq-accordion-content-inner');
                var h = inn.scrollHeight;
                
                if (c.classList.contains('open')) {
                    c.classList.remove('open');
                    if (ic) ic.classList.remove('open');
                    c.style.height = '0';
                } else {
                    var scrollY = window.scrollY;
                    c.classList.add('open');
                    if (ic) ic.classList.add('open');
                    c.style.height = h + 'px';
                    requestAnimationFrame(function() {
                        window.scrollTo(0, scrollY);
                    });
                }
            });
        }
    }
    
    if (document.readyState === 'loading') {
        document.addEventListener('DOMContentLoaded', initGkqAccordions);
    } else {
        initGkqAccordions();
    }
    
    window.initGkqAccordions = initGkqAccordions;
})();

            <p>8月29日写的，被知乎吞了，现在重新写一份，语言上更加简洁一些。</p>
            <p>这一系列是人民币的"准货币"系列，主要是权威机构发行的、在一定区域或时期提供流通职能的钞票，或是国家级无记名债券。债券里包括国库券、国家经济建设公债，地方经济建设公债、国家保值公债、重点建设债券等，类货币包括军用代金券、外汇兑换券、人民银行本票等。预计2021年7月1日之前写完。</p>
            <p>国库券是最常见的一种国债，在1981-1997年间发行，最后一期于2000年到期，从此国内实物国债消失。国库券永不作废，每年财政部公告都会注明实物债券继续兑付，只是国库券逾期不计息，到期后只以计息期间的本利兑付。很多人家里可能都会有几张当年发工资时候买的五块十块的。</p>
            
            <div class="gkq-img-single">
                <img src="readmes/image/gkq/zhihu_headphoto_1.avif" onerror="this.style.display='none'" alt="国库券">
            </div>
            
            <h4>一、国库券的面值</h4>
            <p>在1981年有十元，百元，千元，万元，十万。1982年1、5、10、100。1983-1991年间5、10、50、100，1990、1991有20元面值。1992年的50、100、500、1000。1993-1994有100、500、1000。1995-1997有100、1000、5000元。只有100元是每年都有发行的，直到今日电子化的债券也是以100元为"一张"来交易。</p>
            
            <h4>二、国库券的主题</h4>
            <p>多种，没有固定模式，经济建设、工农业生产、风景名胜是最常见的。1990年有三张是北京亚运会场馆，1991年全是风景，可以勉强算是主题统一的，其余几乎无规律。</p>
            
            <h4>三、国库券的印刷方式</h4>
            <p>1981-1987年间的50、100元是凹版印刷，1992和96、97年的100元，以及所有500元及以上的国库券是凹版印刷，其余全是平版印刷。整个1988-1991年的全部是平版。</p>
            
            <h4>四、国库券的纸张</h4>
            <p>火炬五星水印：和外汇兑换券相同，1981年国库券</p>
            <p>古钱水印：和第四套人民币1980版1、2、5元纸币相同。1990-1992年的50元及以上面值，其中1990年的古钱水印中带有PBC字母，中国人民银行英文缩写，并不是财政部。</p>
            <p>GKQ水印：1993年后的所有国库券。其中，1995五千，1996一千和五千，1997一千和五千带有彩色纤维。这种水印为国库券独有。</p>
            <p>尺寸在1983年之后只有两种：5、10、20的小尺寸，50、100、500、1000、5000的大尺寸。</p>
          
            <h4>五、国库券的收藏方向</h4>
            <p>按面值只有5、10、20方便收集全套。还有个1元单独成套。50元在1987年之前都很贵。100元在1990年之前都很贵，1996 100元也很贵。500元只在1992、1993、1994有发行，1994二期，1992二期很贵。1000元在1981、1992之后有发行，1992、1995很贵。5000元及以上全都是高档品种。</p>
            <p>按主题不太好说，因为啥主题都有很贵的东西，不说也罢。</p>
            <p>按年份，1991年最便宜，其次是1990年，其余每个年份至少都要近万元或以上的预算了。1991年是唯一实钞价格低于票样的，国库券大多高档品种都是票样更便宜。</p>
            
            <h4>六、收藏建议</h4>
            <p>低端就是1982-1991年十年间的20元及以下面值是个小全套，若是再加上1992-1997一百元各一张，即可以收集成小套。</p>
            <p>中端可以再加上1988-1992年的五十元，1991-1995年的一百元细分品种。</p>
            <p>中高端可以在加上部分500、1000元，以及1996二期和1990的100元。比如1993、1994、1996、1997的一千元就不算太贵，几张500元除了1994二期也都不离谱。</p>
            <p>更高端包括了1982-1989的一百元，1983-1987的五十元，这几张一百元都是万元或以上的价格，1983-1985的五十元也不弱。1995千元，1992五百元二期，1992千元，1997五千。</p>
            <p>特高端包括了1981的万元、十万元，1994五百、一千的二期，1995五千东方明珠，1996两张五千，1983、1985的百元，这几个有些都是有钱也不太好买的东西了。</p>
            
            <h4>七、逐年分析</h4>
            <p>1981年的国库券原则上是对公发行的，所以10、100这两张都比不过1982年的，早起国库券在一些单位、国企之类的档案里还时有发现，一旦流出来一些就对价格有冲击。不怎么建议碰这一年的国库券，1981年10元就没见过几张旧的。</p>
            <p>1982年的国库券也是横式，这两年的国库券都和五十年代的债券有些类似，1982年有张1元，为国库券中的唯一，这一年的5、10都是小面值中绝对的龙头，当然100元宝钢也不是吃素的，但是价格比不上83、85的。</p>
            <p>1983到1989年的规格统一，都是5、10、50、100四种，从此国库券尺寸统一，50元以下小尺寸，50元及以上大尺寸，一直到5000元都是一样大的。5元里价格大概是83、84、86、85、87、89、88，10元里大概是83、85、84、86、87、89、88，50元里大概是83、85、84、86、87、89、88，100元里大概是83、85、84、86、87、89、88。注意50元里86及以后比之前便宜很多，100元里再便宜都是万元级别的。注意1985往往比1984贵，这一年除了大量发行的5元，其余面值都不多。10元方面它和83也有的一比，我有很多连号85十元品相都有很多问题。国库券这个问题很严重，因为几乎没有库存，所有的都是使用过的，一旦评级出高分都不便宜。还有1989年或许是发行保值公债太多，各种面值比起1988年似乎是反而少了一点点，1988年是国库券开始交易的年份，这一年50元发行量大增，并且期限变作3年，所以比1987年档次都要低。</p>
            <p>1990、1991是国库券中风格较为统一的，1990年北京亚运会，这一年的5、20、100票面上都是亚运会场馆，算是国库券中难得的结合时事的纪念版本，虽说这一年的设计都不好看。1991是最便宜一年，同时也是最漂亮一年，风格统一，全是山水，最适宜全套收集。这两年都有20元，也都不贵。</p>
            <p>1992年之后的国库券都有些复杂了。</p>
            <p>1992年分作两期发行，最低面值50元，第一期五年，第二期三年，利息还差不远，导致当年有涂改现象。这两期看正面的冠号能看出来，背面多个第二期字样，其余完全一样。50、100元都是第一期更少，500第二期更佳，1000元只有第二期，是千元票中的中高档品种。</p>
            <p>1993年国库券设计较为现代化，用了GKQ水印专用纸张，防伪技术有所升级。这一年利率加上保值补贴率利息特高，三年百分之六十多，五年百分之七十多，当年的利率很是厉害了。但是这不妨碍1993年的各种面值都不出彩，但是普遍五年期少于三年期。这一年吸取了教训，两期改变了颜色。</p>
            <p>1994年保持了1993年的形制，100元又是最便宜的百元之一，这一年虽说推出了两期，但是没了保值补贴，期限大减。虽说装模作样发行了两期，利息期限完全一样的，1994年第二期大大少于第一期，第二期背面都是蓝色，即使是蓝100都是远远少于棕100，蓝500、1000为高档品种，远远贵于93，所幸票样不贵。</p>
            <p>1995年很简单，是国库券总印刷数量最小的一年，虽然都印刷了第一期，实际上也就一期，这一年取消500，增加5000。1000元布达拉宫是国库券设计之最，抢跑了第五套人民币的五十元图案，本身这一年的设计也很大胆。这一年的一千无论设计价格都是精品，比前比后都胜出。这一年的5000图案是东方明珠，当然了，升值率比起东方明珠的房价也是不示弱的，为后期国库券最大珍品。</p>
            <p>1996在后期是最复杂的，一期二期的利率、图案都不一样，二期100元是九十年代100元最贵，这一年开始国库券全部用凹印了。1996千元不贵，两张都还算漂亮，我也刚买了一张牧场，毕竟是人民币币王的主题。两张五千都是大精品，可遇不可求。都得五万左右了吧，这种东西本就不多见，其实是没有公允价格的。</p>
            <p>1997是最后一年了，这三张的图案名气都变大了，外高桥，京九铁路，北京西站，难得一年三个图案名气都大。这一年有钱都能买到，五千元也不算多么贵，一万多即可，毕竟面值也那么高了。一千元是最便宜的一张千元，不到两千块就能买到。</p>

            <p>这两年的国库券奠定了往后近15年里国库券的一些形制。</p>
            <p>一是尺寸。1983版奠定了今后国库券的尺寸，只是在1985年以后全部不再使用竖式布局。即小面值（20元及以下）142×64，大面值（50元及以上）156×72，注意！小尺寸国库券大概是三版炼钢的尺寸，大尺寸国库券的尺寸大概是大团结的尺寸。</p>
            <p>二是文字。1984年修改了1981-1983国库券的繁体隶书字体，改为简体字，同时背面的财政部印章有所更改。</p>
            
            <div class="gkq-img-single">

            <p>这两年总的来说是很相似，83年全面都比84年的贵一些，特别8350比8450还是高档很多的，83版多数面值的价格都在该面值中处于数一数二的地位。</p>
            <p>这两年的期限都是5-9年，即第五年开始抽签兑付，每年还一部分，第九年还清。九年之后都是按照72%的利息，再逾期就不计算利息了。个人购买的都是百分之八的年利率。在84之后，个人的国库券都是到期一次还本付息了。</p>
            <p>单位购买的国库券不使用实物券，因为通常金额较大，而采取国库券收款单的形式，并且利率直接缩水一半，只有百分之四。单位的国库券事实上延迟兑付，十年才还清......这也体现了我国始终还是以群众利益为重。</p>

            <p>先说1992年，九二年的国库券发行量是很大的。相对于之前的每一年来说，都不在一个体量上，本身的面值就非常大，并且印刷量也较大，但是它的好处是¥100以上都是凹版印刷，1992年的100元前后几年，往前到1988年，往后到1996年，都是全胶版的。所以1996、1997、1992年这三年的¥100是雕刻版国库券最便宜的品种。</p>
            
            <p>1992年的第一期和第二期国库券的区别只在背面多了一个第二期的加字，以及正面冠号的区别，1000元只有第二期。其余三种面值，50、100元都是第一期更少，500元是第二期更少。第一期期限五年，第二期期限三年，当年有很多把第一期涂改成第二期的状况出现。为此，1995年左右为此多次下发文件通知。</p>

            <p>涂改债券的风波还不算完，1993年，由于物价上涨太快，1992年国库券市场价格低于面值。财政部发公告，调整部分国库券利率，并且开始使用保值补贴。真是阴差阳错，1989年保值公债都没用上。</p>

            <p>97年是国库券发行的最后一年，这一年发行量普遍较大，97年100见过好几次整刀的，补号JZ也不是极度罕见的品种了。1000元有不少大连号存在。5000元也不是高不可及，还是容易买到的，长期收藏这个目前看来是肯定跑不过通胀。</p>

            <p>97年的印刷不论胶版还是凹版质量都已经很高了。背面很少见的多色，之前的几乎都是单色底纹加上红色印章。</p>

            <p>忘了说一点，95年以来的高面值国库券纸张不仅有荧光纤维，还有有色纤维。有色纤维这个技术在99版人民币上用了几年，05版取消了，也确实是用处不大。但是市场上还是有不少假的京九铁路存在的，印刷垃圾。国库券假票很容易识别，对于收藏没有很大风险。</p>
            <p>从1997之后，实物国库券就彻底消失了，1998年国库券网上有图片，但是没有发行。同样，1998年国库券又是个保守的设计，仍然没有脱离双数年份的风格。</p>
